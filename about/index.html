Why?

It has been nearly one decade since I stopped updating public Blogs hosted on other sites. Looking back into the journey to today’s TULIP Lab, there are many interesting experiences and valuable lessons that worth to be shared or recorded.

I would like to settle down here for a self-controllable Blog hosting site, and wish all the best!

How?

Packages and Services

This blog is built on the top of the following services and tools:

  • Hexo: static blog framework
  • Hexo-Next: my selected theme, with setup document here
  • Hexo-Next-Canvas
  • Hexo-Blog-Encrypt
  • Markdown
  • Real Favcon Generator: icon generator
  • Livere: comments tracking
  • GitHub: hosting service

Trouble Shooting

Most issues come from imcompatable packages.

If upgrading or installation does not work or hexo list post can not identify existing posts, you can initiate a new site in a different folder, copy the git respository files there. Maybe need to re-create configurations or package.json. Once working normally, you can copy the new site files back to your git repository folder.
